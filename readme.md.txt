# Road Accident Dashboard (Power BI)

This project contains an interactive Power BI dashboard that visualizes road accident data  
It showcases trends, severity, and other insights to assist in decision-making.

## Tools Used
- Ms Excel
- Power BI
- Data Visualization
- Data Cleaning and Transformation

## How to Use
1. Download the `.pbix` file.
2. Open it using Power BI Desktop.
